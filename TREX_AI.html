TREX AI

Two versions of the dino game were used, the original google one and a copy built in JavaScript.

A JavaScript copy was used to train the genetic algorithm as it can be run faster than real time, reducing training time significantly. Trained weights were copied to a python script which played the original game.

Using the original game, an object detection algorithm was trained to find the positions of the trex and obstacles. These positions were fed into another neural network, which contains the weights obtained from the JavaScript version, and can play the game in real time.
