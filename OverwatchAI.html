Overwatch Aimbot

This project utilized object detection to create an aimbot for overwatch. First the Yolo V2 algorithm was trained to detect the bots. Pretrained weights were used to speed up training/accuracy due to low amounts of labeled data of the bots.

Once the algorithm has been trained, it is used to determine the positions of bots on screen and the bot closest to the center is deemed the target denoted with a red circle inside the green detection box. The direction vector is calculated, which is then used to move the joystick to the desired location.

A lot of limitations arise with this aimbot. First, it can only run at frame rates upto about 20 fps. This is becuase the game is running at 75fps and the neural network can only use the remaining GPU memory. Furthermore, the joystick emulation does not allow very small movements, so the joystick often overshoots.
