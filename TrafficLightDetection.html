Traffic Light Detection

This project takes the output of the YOLO object detection, which only detects if a traffic light is present, and classifies weather it is red, yellow or green. The aim of the project was to integrate it with the self driving car to give it more information about the world.

To obtain training data, YOLO object detection was used to gather images of traffic lights, then each image was classified using a color picker. Any inaccuracies with the colour picking were discarded by hand. The remaining images were used to train a convolutional neural network to classify the traffic light.

In practice, only the traffic light that is closest to the horizontal center is used as this should be the traffic light in our lane.
