Self Driving Car Vision

This project is the backbone for a future upgrade to the GTA self driving car project. Currently the GTA project simply uses an end to end network to predict steering and throttle commands from an input image. The aim with this project is to use the lane, object detection and depth perception to create a more consistant self driving car.

The vision model currently consists of 3 separately trained models (lane, segemntation and depth), which have been combined to create 1 end to end vision model. It was trained on open source datasets, with a combination of real and synthetic datasets (from games )to try and improve generalisability.

For even more information about this project, visit the GitHub page https://github.com/Will-J-Gale/Self-Driving-Car-Vision
