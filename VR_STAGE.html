VR Stage Interface

Building on from the LAMI interface, this project aimed to solve the problems that arose from that interface using a Gear VR interface instead of a screen. These problems included clutter, depth perception object occlusion, depth perception, object interaction, exit error and gorilla arm.

This project was published in the WIMP 2018 conference: https://research.hud.ac.uk/media/assets/document/research/2-WillGale-WIMP2018-VR.pdf
