GTA5 Self Driving Car

This project was heavily inspired by Harrison Kinsley's project: https://github.com/Sentdex/pygta5

This model was trained using the behavioural cloning technique. Approximately 80,000 images were collected along with the corresponding steering angles and throttle positions. This training data was then given to the model to learn how to drive.

A modified version of PilotNet was used for the model, with one main difference being that there was two regression heads instead of one, one for the steering angle and one for the throttle.

Overall the model mainly learns to steer, and maintain speed at about 45mph. With more data the model could possible generalise further. However, with the addition of depth perception, optical flow estimation and scene segmentation this could help the model learn better representations with similar amounts of training data.

For even more information about this project, visit the GitHub page https://github.com/Will-J-Gale/GTA5-Self-Driving-Car
